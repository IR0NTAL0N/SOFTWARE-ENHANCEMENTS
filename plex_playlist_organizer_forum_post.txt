Title:
Plex Playlist Organizer for Windows - Fix Huge Plex Playlists Without Fighting the Plex GUI

Body:
I built a Windows desktop tool for managing very large Plex playlists because Plex’s native playlist GUI is painfully slow and cumbersome once your list gets big.

What it does:
- Move single titles
- Move blocks of titles
- Search and jump through matches
- Export playlist order to CSV
- Import edited CSV back into Plex
- Backup and restore playlist order
- Copy playlists from one Plex server to another
- Works with multiple servers under one admin account

Why it exists:
If you have a huge Plex playlist, especially one with franchise groupings or carefully curated order, Plex makes reordering it a miserable experience. This tool fixes that.

Basic setup:
1. Put these files in one folder:
   - plex_playlist_organizer_gui_v8c.py
   - launch_playlist_organizer_gui_v8c.bat
2. Install Python on Windows
3. Run the BAT file
4. On first launch, enter:
   - Global Plex token
   - Backup root folder
   - Server name
   - Server URL (example: http://192.168.1.21:32400)
5. Save config and use it

Main features:
- Search box at the top with Next / Prev
- Block cut/paste for moving multiple titles at once
- CSV round-trip for editing playlist order in Excel
- Backup and restore safety net
- Cross-server playlist copy by title matching
- Optional suppress-confirmations setting

Important:
- Use the full server URL, including http://
- This is a single-admin tool using one Plex token
- Backups include server name in the filename
- Best for people managing large curated playlists
